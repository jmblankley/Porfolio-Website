The Casters Companion

Description

Independently conceptualized, designed, developed, and deployed "The Casters Companion," a Dungeons and Dragons (DnD) character companion app tailored for managing castng abilities. The project served as an opportunity to learn and implement the MERN (MongoDB, Express.js, React, Node.js) stack in a real-world application.

Role

Sole Developer, Designer, and Project Manager

Responsibilites

  • Designed and developed the user interface and experience for the app, ensuring user-friendly navigation and efficient casting management.
  • Implemented the MERN stack, building a responsive front-end with React and managing data with MongoDB and Express.js.
  • Created APIs and integrated the back-end functionalities using Node.js, enabling seamless data handling and manipulation.
  • Conducted rigorous testing and debugging to ensure optimal performance and a smooth user experience.

Challenges

Overcame the challenge of learning and implementing the MERN stack, leveraging this project as a practical way to grasp its components and their interaction

Outcome

Ongoing development of the app has resulted in a powerful tool for DnD enthusiasts, streamlining casting management for characters. The beta app is available as a PWA now!

Technologies Used

HTML, CSS, Javascript, React, MongoDB, Express, Github, Linux, Node.js
